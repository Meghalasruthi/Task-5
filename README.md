# Task 5: Capture and Analyze Network Traffic Using Wireshark

## Objective
To capture live network traffic and identify basic network protocols using Wireshark.

## Tool Used
- Wireshark (Free)

## Procedure
- Installed Wireshark on Windows
- Started packet capture on active Wi-Fi interface
- Generated traffic by browsing websites
- Stopped capture after sufficient packets were captured
- Applied protocol filters like DNS and TCP
- Saved the capture as a .pcap file

## Protocols Identified
- DNS – Domain Name System
- TCP – Transmission Control Protocol
- TLS / HTTP – Web communication protocols

## Outcome
Successfully captured and analyzed network traffic and gained hands-on experience in packet analysis.
